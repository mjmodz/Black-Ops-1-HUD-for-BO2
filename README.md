# Black Ops 1 HUD, Scoreboard & Pause Menu for Plutonium T6

Hey everyone!

Today I’m releasing my `Black Ops 1 HUD, Scoreboard, and Pause Menu` for `Call of Duty: Black Ops II - Plutonium T6 Zombies`.

This is my first ever public release, so I hope you guys enjoy it!

## Features

- Black Ops 1 HUD
- Black Ops 1 Scoreboard
- Black Ops 1 Pause Menu

## How to Install

Download: [zm_t5_hud](https://github.com/mjmodz/Black-Ops-1-HUD-for-BO2/releases/tag/v1.0.0)

1. Go to your Plutonium T6 folder.

   Press `Win + R`, paste this `%localappdata%\Plutonium\storage\t6`, then press `OK` or `Enter`:

2. Open the ZIP file you downloaded.

3. Move the `zm_t5_hud` folder into your T6 `mods` folder, if you do not have a `mods` folder, create one first.

4. Launch Black Ops II through Plutonium.

5. Load the mod from the in-game Mods menu.

That’s it. Load into Zombies and enjoy! 😊
